### 无限脚手架
[链接](https://github.com/CakeDreamer/SMP2.plugins/tree/main/InfiniteScaffold)
* 可无限放置的脚手架生成器，生成的脚手架破坏后不掉落
* 与玩家绑定，非持有者不能使用
* 监听自定义成就树完成事件，发放脚手架与其他奖励（执行其他插件指令）
